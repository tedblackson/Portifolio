EVENTO

Evento is an event organizing platform (Web App) that has two types of users the first types of users are event organizers(event host) and the second types of users are event attendees which are normal that are currently searching events that is favoriable for them.

    Features

  • Event attendees can access suggested events based on a recommendation form they fill.
  • Events attendees can search for events
  • Event attendees can access events in a sorted order of the distance from their living address

    Technologies Used

  • HTML
  • JAVA ( Spring Boot)
  • CSS3

MEGENAGNA

Megenagna is employee to an employer connecting plarform, where there are two types of accounts one is an account for the company's with a job offer and the second one is for individuals who are looking for a job

    Features

  • Individuals can search for a job using different keywords
  • Company's can access the statistical analaysis of the people that applied to the job they posted

    Technologies Used

  • Flutter
  • Python(Django)
  • Dart
